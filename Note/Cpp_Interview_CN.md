###### 1.1. C 和 C++ 的区别
  
设计思想上:
- C++ 是面向对象的语言, C 是面向过程的语言. [Case](https://www.zhihu.com/question/24425316)

语法上:
- C++ 具有封装/继承/多态三种特性.
- C++ 相比 C, 增加了类型安全的功能, 比如强制类型转换.
- C++ 支持范式编程, 比如模板类/函数模板等.
